Prerequisities
--------------

A supported OS (tested with Linux).
LAME library and header(s) (tested with version 3.89).
C compiler (tested with gcc 2.95.4).

Lametus requires Qt development libraries. Install them
using your distributions tools. 

On Debian and Ubuntu this should be enough:

apt-get install libshout3-dev libmp3lame-dev libqt4-dev

Compiling
---------

Just run:

qmake
make

Installing
----------

Just copy the "lametus" binary to whatever place you see fit.
